# Технічне завдання — `full_system/`

## Мета

Модульний Octave-каркас для курсової роботи **«Порівняння ефективності та обчислювальної складності класичних і нейромережевих алгоритмів MPPT в умовах мінливої освітленості»**.

Каркас має:
- бути читабельним для студента (зрозумілий код, коментарі українською);
- давати готові сценарії досліджень (свіпи параметрів);
- демонструвати щонайменше один сценарій, де NN виграє, і один, де P&O впевнено кращий.

## Параметри (узгоджено)

- **Координати:** Львів (49.83°N, 23.96°E)
- **Панель:** Kyocera KC200GT (200 Вт, Voc=32.9В, Isc=8.21А)
- **Масив:** 10 послідовно × 2 паралельно = 4 кВт
- **Мова:** українська для всього (коментарі, console output, документація)
- **Без Simulink** — лише Octave-скрипти
- **Pure Octave**: жодних додаткових пакетів (`nnet` тощо)


## Архітектура нейромережі (узгоджено)

- FC: **2 → 12 → 8 → 1** (всього 145 параметрів)
- Активації: tanh у схованих, лінійна на виході
- Вхід: [G/G_norm; (T-T_shift)/T_norm]
- Вихід: V_mpp/Voc_array (нормалізований)
- Тренування: SGD, lr=0.02, 800 епох, batch=64
- Backprop — вручну з коментарями
- Датасет: 3000 синтетичних зразків від `pv_mpp`

## Особливості, узгоджені під час розробки

1. **Симуляція з кроком 1с**, P&O оновлюється раз/секунду — реалістично для real-time трекера в інверторі.
2. **t_hours як локальний сонячний час** (t=12 → полудень). Координати залишаються як параметри для майбутнього розширення.
3. **Кешування MPP** — `precompute_mpp_grid` робить lookup-таблицю по (G, T) для прискорення симуляції.
4. **Хмари — марковська модель** з 2 станами + згладжування переходів, замість незалежних бернуллі.
5. **Графіки**: рендеряться через gnuplot toolkit (працює в headless). У Octave GUI кирилиця відображається коректно; у CLI — частково (обмеження gnuplot PNG).

## Очікувані результати (з тестів)

- P&O: ~98-99% від ідеалу (типовий день, хм. 40%)
- NN: ~99-99.6% від ідеалу (тренована мережа, RMSE V_mpp ~5%)
- NN/P&O cost ratio: ~2.5-3x за виклик
- Енергетичний виграш NN над P&O: помітний при високій мінливості G
