# MPPT Системи: Порівняння Класичних та Нейромережевих Алгоритмів

## Загальний опис

Цей проект реалізує та порівнює дві стратегії відслідження точки максимальної потужності (MPPT) сонячної панелі:

1. **Класичний алгоритм P&O (Perturb & Observe)** - простий, але повільний та неоптимальний у нестійких умовах
2. **Нейромережевий MPPT** - навчена нейронна мережа для прямого прогнозування оптимальної напруги

## Структура проекту

```
full_system/
├── docs/                    # Документація
├── solar_model/            # Модель сонячної панелі KC200GT
├── cloud_model/            # Модель варіацій хмар
├── mppt_classical/         # Класичний P&O алгоритм
├── neural_network/         # Нейронна мережа
├── data_generation/        # Генерація даних для тренування
├── simulation/             # Основні симулятори
└── analysis/               # Інструменти аналізу
```

## Параметри системи

| Параметр | Значення |
|----------|----------|
| **Панель** | KC200GT |
| **Масив** | 10×2 = 20 панелей послідовно-паралельно |
| **Локація** | Львів (49.8°N, 24.0°E) |
| **Діапазон освітленості** | 0-1000 W/m² |
| **Діапазон температури** | 15-60°C |

## Нейронна мережа

Архітектура: **2-8-4-1** повнозв'язна мережа
- **Входи**: Освітленість (W/m²) + Температура (°C)
- **Приховані шари**: 8 → 4 нейрони (tanh активація)
- **Вихід**: Оптимальна напруга (V)

## Метрики порівняння

1. **Вироблена енергія** - загальна енергія за період симуляції
2. **Ефективність відслідковування** - відносна потужність до теоретичного максимуму
3. **Швидкість відгуку** - як швидко алгоритм досягає оптимуму
4. **Обчислювальна складність** - кількість операцій
5. **Стабільність** - коливання напруги біля оптимуму

## Проблеми класичного P&O

Проект демонструє типові проблеми класичного P&O:
- **Низька швидкість** відслідження при швидких змінах освітленості
- **Неоптимальність** під час хмарних днів
- **Осциляції** навколо оптимальної напруги
- **Залежність від кроку** параметра від умов

## Переваги нейромережевого підходу

- **Швидкість**: Прямий прогноз без ітеративного пошуку
- **Адаптивність**: Натренована на різних сценаріях
- **Прозорість**: Легко модифікувати для нових умов
